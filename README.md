# 📘 PLN com Machine Learning — Da Teoria à Prática

Repositório oficial do mini-curso **PLN com Machine Learning — Da Teoria à Prática**.

Este curso foi desenvolvido com o objetivo de apresentar, de forma integrada, os fundamentos de **Processamento de Linguagem Natural (PLN)** e **Aprendizagem de Máquina (Machine Learning)**, conectando teoria e implementação prática em Python.

A proposta é demonstrar como dados textuais podem ser tratados, representados numericamente e utilizados na construção de modelos preditivos aplicáveis a problemas reais.

---

## 🎯 Objetivos do Curso

- Compreender os fundamentos de PLN
- Entender como textos podem ser transformados em representações numéricas
- Aplicar técnicas de Machine Learning em tarefas envolvendo dados textuais
- Construir e avaliar modelos de classificação
- Integrar teoria e prática em um fluxo completo de análise

---

## 🧠 Conteúdo Abordado

O mini-curso percorre as principais etapas de um projeto de PLN com Machine Learning:

- Preparação e limpeza de dados textuais  
- Técnicas de pré-processamento  
- Representação vetorial (ex.: Bag of Words, TF-IDF)  
- Treinamento de modelos supervisionados  
- Avaliação de desempenho e interpretação de resultados
